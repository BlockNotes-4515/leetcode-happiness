# Contributing

Thank you for your interest in contributing!

## How to Contribute

1. Fork this repository.
2. Create a new branch.
3. Add or improve Java solutions.
4. Follow clean coding practices and include comments where necessary.
5. Test your solution before submitting.
6. Commit your changes with a meaningful message.
7. Open a Pull Request describing your changes.

## Guidelines

- Write solutions in **Java**.
- Keep the code clean, readable, and optimized.
- Use descriptive file names matching the problem name.
- Do not submit duplicate solutions unless they demonstrate a different approach.
- Ensure your code compiles successfully.

## Reporting Issues

If you find an error or have suggestions for improvement, please open an issue with relevant details.

Thank you for helping improve this repository!
