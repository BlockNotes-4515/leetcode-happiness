# Security Policy

## Supported Versions

This repository is actively maintained, and security updates apply to the latest version of the code.

| Version | Supported |
| -------- | --------- |
| Latest   | ✅ Yes |
| Older versions | ❌ No |

## Reporting a Vulnerability

Although this repository primarily contains Java solutions to LeetCode problems and is not production software, responsible disclosure is appreciated.

If you discover a security issue or believe there is a vulnerability related to the repository, please:

1. Open a GitHub Issue (for non-sensitive concerns), or
2. Contact the repository owner privately if the issue contains sensitive information.

Please include:
- A clear description of the issue.
- Steps to reproduce (if applicable).
- The potential impact.
- Suggested remediation (optional).

Thank you for helping keep this repository secure.
